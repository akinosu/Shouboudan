# 消防団.com
 https://shouboudan.tk/index
 
"消防団.com"は移住者のための、全国各地の消防団に関するクチコミサイトです。
 ![消防団.com](screencapture-shouboudan-tk-index-2022-02-05-19_37_36.png)
 
# 概要
 
"hoge"の魅力が直感的に伝えわるデモ動画や図解を載せる
 
 
# 背景
 
"hoge"のセールスポイントや差別化などを説明する
 
 
# 使用技術
 
* PHP 8.0.13
* Laravel 6.20.43
* MySQL 8.0.28
* Nginx 1.20.0
* AWS
    -VPC
    -EC2
    -Route53
* Twitter API
* JapanMap.js
